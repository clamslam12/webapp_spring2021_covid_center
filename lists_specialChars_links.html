Lists are block elements. Unordered (ul) list, ordered list (ol), description list (dl)

Some important attributes for ordered lists Start=‘10': Start the list from 10.....; Reversed

You can also nest lists within lists.

Simple ordered list

  1. milk
  2. eggs

Ordered list with attributes start="2" and reversed

  1. juice
  2. cookies

Nested lists: ordered list and description list nested inside unordered list

  • item 1
  • item 2
    1. item 2.1
    2. item 2.2
  • coffee
    hot black drink
    milk
    cold white drink
